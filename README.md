
# 🎉 Happy Birthday Surprise Page 🎂

A fun, interactive, and beautiful HTML-based birthday greeting webpage made with love and style! This page features:

- Animated text and hover effects
- Confetti, balloons, hearts, and stars floating animations
- A personalized birthday greeting message
- Background music with play/pause and volume controls
- Typewriter animation and transition to a surprise page

## 🚀 Live Demo : https://happy-birthday-surprise-page.vercel.app

Just open `index.html` in any modern browser.


## 🎨 Features

- 💖 **Responsive Design**: Mobile-friendly layout
- 🎵 **Background Music**: Auto-play with controls
- ✨ **GSAP Animations**: Smooth, lively animations
- 🎈 **Decorative Elements**: Confetti, hearts, balloons, and stars
- 🖱️ **Interactive Button**: Animated navigation to `memorylane.html` (you can customize this)

## 🛠️ Tech Stack

- HTML5
- CSS3 (with custom animations)
- JavaScript (Vanilla JS)
- GSAP (GreenSock Animation Platform)


## 📦 How to Use

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. (Optional) Customize the text, colors, or music.
4. (Optional) Add your `memorylane.html` or any additional content/pages.

## 💡 Tips

- Host it on GitHub Pages, Netlify, or Vercel for free sharing.
- Use your own audio file if you want to play a custom birthday song.
